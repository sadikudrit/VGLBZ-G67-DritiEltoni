## Sistemi Smart Library - Grupi G67
Bashkëpunëtorët: Drit Sadiku & Elton Maloku
# Përshkrimi i Projektit
Ky projekt është një aplikacion menaxhimi për biblioteka, i zhvilluar në gjuhën C++. Ai mundëson regjistrimin e librave në një inventar digjital dhe procesimin e huazimeve në kohë reale. Sistemi është ndërtuar duke u fokusuar në programimin e orientuar në objekte (OOP) dhe menaxhimin e të dhënave përmes strukturave vector.
# Kriteret e Realizuara
Sipas kërkesave të projektit, ne kemi zbatuar:
Version Control: Përdorim i rregullt i Git për commits, push dhe pull.
Bashkëpunimi: Punë paralele në degë të ndryshme (driti dhe eltoni) dhe bashkim i kodit përmes Pull Requests.
Dokumentimi: Ky skedar README dhe komentet brenda kodit burimor.
# Veçoritë Kryesore
Shtimi i Librave: Regjistrimi i titullit, autorit dhe një ID-je unike.
Huazimi i Librit: Ndryshimi i statusit të librit dhe regjistrimi i emrit të huazuesit.
Gjenerimi i Datës: Përdorimi i librarisë <ctime> për të shënuar kohën e saktë të huazimit.
Shfaqja e Inventarit: Prezantimi i të gjithë librave në formë tabele të formatuar.
# Si ta ekzekutoni projektin
Klononi repository-n:
Bash
git clone https://github.com/sadikudrit/G67-Driti-Eltoni-
